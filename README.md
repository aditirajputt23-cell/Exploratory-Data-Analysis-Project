# Exploratory Data Analysis (EDA) Project

## Overview

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to identify trends, patterns, and key factors influencing passenger survival.

## Objectives

* Explore dataset characteristics
* Generate statistical summaries
* Identify patterns and correlations
* Visualize important trends
* Present findings and insights

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Dataset

Titanic Dataset

## Analysis Performed

* Dataset inspection
* Statistical summary
* Missing value analysis
* Survival distribution analysis
* Gender-based analysis
* Passenger class analysis
* Correlation analysis using heatmaps

## Key Findings

* Female passengers had higher survival rates.
* First-class passengers were more likely to survive.
* Passenger class significantly influenced survival outcomes.
* Most passengers were adults.
* Fare and class showed meaningful relationships with survival.

## Conclusion

EDA helped uncover important trends and relationships within the Titanic dataset, demonstrating the importance of data exploration before building machine learning models.
